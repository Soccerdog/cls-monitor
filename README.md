# 财经新闻监控脚本

自动监控东方财富财经要闻，当检测到**重要新闻**时，通过飞书实时推送通知。

> ⚠️ 原财联社接口已失效，现已切换为东方财富要闻 API

## 功能特性

- ⭐ **智能筛选**：只推送官方推荐的重要新闻（红色标记）
- ⏰ **定时检查**：每分钟自动检查最新财经要闻
- 📢 **飞书推送**：重要新闻即时推送到飞书对话框
- 🔄 **去重机制**：已发送的新闻不会重复推送
- 💤 **免打扰**：支持设置每日免打扰时间（默认 23:00 - 次日 08:00）
- 🚀 **开箱即用**：纯 Node.js 实现，无需额外依赖

## 安装与运行

### 1. 环境要求

确保你有 Node.js 环境（建议 v14+）：

```bash
node --version
```

### 2. 启动监控

```bash
cd cls-monitor
node index.js
```

脚本会自动通过 OpenClaw 发送消息到你当前对话的飞书用户。

### 3. 后台运行

```bash
# 使用 nohup
nohup node index.js > cls-monitor.log 2>&1 &

# 或使用 PM2（推荐）
npm install -g pm2
pm2 start index.js --name cls-monitor
pm2 logs cls-monitor
```

## 配置选项

可在 `index.js` 中修改以下配置：

| 配置 | 说明 | 默认值 |
|-----|------|--------|
| `CHECK_INTERVAL` | 检查间隔（毫秒） | 60000 (1分钟) |
| `DND_START` | 免打扰开始时间（小时） | 23 |
| `DND_END` | 免打扰结束时间（小时） | 8 |

### 修改免打扰时间

编辑 `index.js`：

```javascript
const DND_START = 23; // 23:00
const DND_END = 8;    // 次日 08:00
```

### 修改检查频率

```javascript
const CHECK_INTERVAL = 30 * 1000; // 30秒检查一次
```

## 新闻筛选规则

只有满足以下条件的新闻才会推送：

- `titlestyle === '3'` - 东方财富官方推荐的红色标记新闻

## 数据文件

脚本会自动创建 `sent_news.json` 文件用于存储已发送的新闻 ID，避免重复推送。

**注意**：删除此文件会导致之前发送过的新闻重新推送。

## 常见问题

### Q: 消息发不出去怎么办？

A: 确保 OpenClaw Gateway 正在运行，且飞书渠道已配置。

### Q: 如何测试脚本？

A: 启动后观察日志输出，确认有「监控已启动」提示。

### Q: 可以推送到飞书群吗？

A: 目前脚本配置为推送到私聊对话框。如需推送到群，可以修改 `sendFeishuMessage` 函数中的 `--target` 参数。
